Links

  • Formulario html, el formulario clásico usado en la web, no nos permite manejar la respuesta del servidor.
  • Formulario json, utilizando fetch y javascript, nos permite manejar la respuesta del servidor.
  • HTTP Auth, formulario de autenticación con HTTP.
  • JSON Auth, formulario de autenticación con JSON y fetch.
  • Ping, ejecutar comandos de terminal desde el navegador.
  • Figlet, convertir texto a arte ascii.

API

Ejemplo de API RESTful emulando una base de datos de usuarios.

GET /apiv1/users

Obtiene la lista completa de usuarios

localhost:port/apiv1/users

GET /apiv1/users/:id

Obtiene el usuario según su id

localhost:port/apiv1/users/1

POST /apiv1/users

Crea un nuevo usuario

localhost:port/apiv1/users?name=nombre&password=contraseña

PUT /apiv1/users/:id

Actualiza el usuario según su id

localhost:port/apiv1/users/1?name=nombre&password=contraseña

DELETE /apiv1/users/:id

Elimina el usuario según su id

localhost:port/apiv1/users/1
